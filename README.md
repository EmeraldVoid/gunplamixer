## Description

GunplaMixer is a Python-based Gundam Breaker 4 randomizer that generates custom Gunpla builds using in-game parts. It randomly selects a head, body, legs, left arm, right arm, and backpack to create unique mobile suit combinations.

This project was built as a fun passion project for Gundam and Gunpla fans, streamers, and anyone wanting creative build inspiration.

---

## Features

- Contains a list of all the mobile suit kits/parts available in Gundam Breaker 4
- Randomly selects:
  - Head
  - Body
  - Legs
  - Left Arm
  - Right Arm
  - Backpack
- Includes:
  - Fully Random Mode
  - Unique Parts Only Mode
- Save builds as:
  - `.json`
  - `.txt`
- Allows users to name save files
- Includes terminal boot animations
- Includes a `.bat` launcher for convenience and Stream Deck compatibility
- Created using Python

---

## Why The BAT File?

The BAT launcher exists mainly because some systems occasionally have issues running Python scripts directly.

It also allows the software to be launched quickly using hardware such as a Stream Deck, making it convenient for livestreams and content creation.

## Installation

1. Install Python 3
2. Download the repository
3. Place the `.py` and `.bat` files in the same folder
4. Run:

```bash
GunplaMixer.bat
```

**OR** in Windows, double click the `GunplaMixer.bat` file and wait.
