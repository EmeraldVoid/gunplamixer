## Overview

GunplaMixer is a lightweight random build generator for Gundam Breaker 4 designed to create custom Gunpla loadouts using parts available in-game.

Originally created as a personal side project, the application focuses on fast generation, accessibility, and experimentation. It can be used for challenge runs, build inspiration, livestream interactions, community events, or simply creating unusual and unexpected mobile suit combinations.

The project is written entirely in Python and includes multiple interface variants ranging from minimal layouts to retro terminal-inspired designs.

## Purpose

GunplaMixer was created as a lightweight companion tool for players of Gundam Breaker 4 who enjoy experimenting with custom builds, challenge runs, and unconventional part combinations.

One of the core appeals of Gundam Breaker is the ability to freely customize mobile suits using collected parts. GunplaMixer expands on that idea by introducing randomized generation, encouraging players to create builds they may not have otherwise considered.

The project can be used in several ways:

- Generating random builds for challenge runs
- Creating inspiration for future Gunpla designs
- Encouraging experimentation with unfamiliar parts or kits
- Adding variety to gameplay sessions and livestreams
- Creating community challenges or randomized tournaments

Generated builds can also be exported and shared for later use or community events.

## Why This Project Was Made

GunplaMixer began as a personal side project inspired by both Gunpla customization systems and retro terminal-style software aesthetics.

The original goal was simple: create a fast and accessible way to randomize mobile suit parts without requiring spreadsheets, manual selection methods, or complicated setup.

Over time, the project evolved into a more refined standalone utility featuring multiple interface styles, export functionality, and streamlined workflows designed for both casual use and livestream integration.

Although the project serves a niche audience, it was built specifically for players who enjoy customization-heavy gameplay systems and community-driven experimentation.

## Installation

1. Install Python 3
2. Download or clone the repository
3. Place the `.py` and `.bat` files in the same folder
4. Run `GunplaMixer.bat`

> [!NOTE]
> Python can be downloaded from the official website:
> https://www.python.org/downloads/. If you're unfamiliar with Python installation, several beginner-friendly tutorials are available on YouTube.
